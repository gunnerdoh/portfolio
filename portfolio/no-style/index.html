Projects

MCP Server For Security Operations Analysts

AI · Security

For my senior capstone I worked with Crest Data to build an AI-powered alert triage system for security operations centers (SOCs). Our system uses an LLM orchestrated through MCP servers to pull alerts from Jira, enrich them with threat intelligence from VirusTotal, classify the risk, and post a structured verdict back into Jira.
